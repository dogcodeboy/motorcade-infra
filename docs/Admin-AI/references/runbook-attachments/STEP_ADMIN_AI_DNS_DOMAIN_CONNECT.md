# DNS & Domain Connect Governance (Motorcade)

## Scope
- Domain Connect manages Motorcade-owned domains only.
- Client domains deferred for future expansion.

## DNS Types
- Internet-facing DNS
- Internal device DNS

## Naming
- Strict enforced templates per device class.
- Names encode function/location.

## Renaming
- Allowed only for permanent relocation.
- Admin AI controlled, step-up auth, audited.

## Hybrid Environments
- Supports cloud + on-premise infrastructure.
