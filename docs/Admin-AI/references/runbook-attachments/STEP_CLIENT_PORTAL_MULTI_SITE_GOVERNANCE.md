# Client Portal Multi-Site Governance (Motorcade)

## Access Eligibility
- Client portal access restricted to decision makers only.

## Account Limits
- 5 decision-makers per org + 3 per site.
- Exceptions require approval below OWNER.

## Multi-Site Model
- Supports centralized or distributed billing/legal.
- Contracts usually site-owned; org-owned supported.

## Site Visibility
- Default: site-scoped access only.
- Cross-site access requires explicit assignment.

## Abuse Prevention
- User limits enforced.
- Temporary overflow with throttling + alerts.
